單元 2：函式與迴圈

函式和迴圈是程式的核心，學會它們你就可以寫出有重複邏輯和功能的程式。

1️⃣ 函式的定義與呼叫

函式可以將程式重複的部分包裝起來，方便呼叫：

function greet(name){
  console.log("Hello, " + name + "!");
}

greet("小明");
greet("小華");

2️⃣ for 迴圈

當你知道要執行固定次數的程式碼時，可以用 for：

for(let i=0; i<5; i++){
  console.log("第 " + i + " 次執行");
}

3️⃣ while 迴圈

當條件為 true 時，就持續執行：

let n = 0;
while(n < 5){
  console.log("n = " + n);
  n++;
}

4️⃣ 小練習

  • 寫一個函式計算兩個數字相加的結果。
  • 用 for 迴圈列印 1~10 的平方。
